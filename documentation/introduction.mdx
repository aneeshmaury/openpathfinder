## Upcoming Features

Here are some of the upcoming features that OpenPathFinder offers:

- Customized Project Recommendation
- Advanced Search Filters
- Community Collaboration tools
- Mentorship Opportunites
- Learning Resources
- Progress Tracking and Feedback


## Tech Stack

- [Next.js](https://nextjs.org/) – framework
- [TypeScript](https://www.typescriptlang.org/) – language
- [Tailwind](https://tailwindcss.com/) – CSS
- [MongoDB](https://www.mongodb.com/) – database
- [NextAuth.js](https://next-auth.js.org/) – auth
- [Stripe](https://stripe.com/) – payments
- [Vercel](https://vercel.com/) – deployments


## Getting Started

To get started with OpenPathFinder, you can do any of the following:

- Open the "https://www.openpathfinder.vercel.app" and click on Login Now button.
- Sign in your account and if not register then register first.
- Update your profile section to get better project recommendation.
- Then a dashboard is open to find the project which is open to contribute.
- You can connect with your mentors also for better guidance.
- Then explore the platform for your better opportunities.
